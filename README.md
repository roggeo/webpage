# Webpage

The web personal page.


## Install

    npm install
    git clone git@github.com:roggeo/webpage.git


## Running

Running front

    cd webpage
    npm start    

Look at [localhost:3000](http://localhost:3000)
    
Now running API server
   
    cd api
    npm start

Look at [localhost:3000](http://localhost:3003)

## Test

    npm test


## Building React project

    npm run build
